# Change Log

All notable changes to the "Services Manager" extension will be documented in this file.

## [0.0.1] - 2025-12-16

### Added
- Initial release
- Support for SQL Server, PostgreSQL, MariaDB, and Redis services
- Service grouping by type (Databases, Caches)
- Expandable service details showing port, username, password, and database
- Pull Docker images for supported services
- Start and stop services with a single click
- Visual status indicators (not pulled, stopped, running)
- Tree view in the activity bar showing all managed services organized by type
- Refresh command to update service status
- Easy to add new services by adding entries to the SERVICES array
