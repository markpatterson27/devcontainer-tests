# Change Log

All notable changes to the "docker-container-manager" extension will be documented in this file.

## [0.0.1] - 2025-12-16

### Added
- Initial release
- Support for SQL Server, PostgreSQL, MariaDB, and Redis containers
- Pull Docker images for supported databases
- Start and stop containers with a single click
- Visual status indicators (not pulled, stopped, running)
- Tree view in the activity bar showing all managed containers
- Refresh command to update container status
